DrawBrusselator
===============

ブラッセレータ(Brusselator)の方程式を差分で計算し位相図を描くプログラム．
リアルタイム処理における負荷を下げるためあえて数値積分を用いずに漸化式で計算しています．
ブラッセレータについては文献[(1)]を参考にしました．なお続きものとして同名の記事[(2)]がありますのでタイトルで検索する場合には注意が必要ですｗ．
プログラムの構造は文献[(3)]の例を参考にしました．
